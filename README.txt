Este proyecto consiste en la realización de un brazo robótico de pequeño tamaño.
El control de movimiento se realizará con un Arduino MEGA 2560.
Las piezas del robot se fabricarán mediante una impresora 3d, y se diseñarán utilizando FreeCAD.
Adicionalmente se utilizará Python para realizar una interfaz gráfica en un ordenador para poder
mandar órdenes al brazo. Si fuese necesario también se pueden realizar los cálculos  de la cinemática
inversa, si el Arduino no tuviese la potencia necesaria para los cálculos.
